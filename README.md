# Interactive Constellation Viewer

A comprehensive, dynamic web application for exploring the night sky. This viewer renders accurate star maps based on your location and time, featuring real astronomical data, interactive constellations, and detailed star information.

## Features

*   **Dynamic Rendering:** View the night sky from any Latitude and Longitude on Earth.
*   **Time Travel:** Adjustable Date and Time controls to see past or future skies.
*   **Accurate Projection:** Uses Stereographic projection to map celestial coordinates (Right Ascension/Declination) to the screen.
*   **Rich Dataset:**
    *   Over 100 prominent named stars with detailed metadata (Magnitude, Distance, Spectral Type, Name Meaning).
    *   2000+ generated background stars for a realistic immersive experience.
    *   Major constellations with stick figures and mythological descriptions.
*   **Search:** Built-in autocomplete search to quickly find stars and constellations (e.g., "Sirius", "Orion").
*   **Interactivity:**
    *   Hover over stars for quick stats.
    *   Click stars or constellations to view detailed information cards.
    *   Zoom (via mouse wheel).

## Usage

1.  **Clone the repository.**
2.  **Open `index.html`** in any modern web browser. No build steps or servers required.
3.  **Controls:**
    *   **Latitude/Longitude:** Enter coordinates manually or use "Use My Location" (requires browser permission).
    *   **Date & Time:** Pick a specific moment or click "Set to Now".
    *   **Search:** Type in the search box to find celestial objects.
4.  **Explore:** Zoom with the mouse wheel. Click on stars to learn more!

## Technologies

*   **HTML5 Canvas:** For high-performance 2D rendering.
*   **Vanilla JavaScript:** Modular ES6+ code with no external dependencies.
*   **CSS3:** Responsive dark-themed UI.
